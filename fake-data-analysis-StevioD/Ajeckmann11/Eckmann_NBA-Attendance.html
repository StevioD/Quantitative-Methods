The dataset contains 10,000 rows and 8 columns. The dataset is designed to be attendance totals for NBA basketball games. The response variable is attendance, which is the number of fans that attended the basketball game on that day. The other columns are:

Weekday; categorical variable of the day of the game

Time; categorical variable of the time of the game

conf; categorical variable signifying non-conference, conference or rivalry game

opp_win_p; categorical variable of win percentage of the away team

opp_all_stars; variable of number of players on away team that are all stars

win_p; continuous variable between 0 and 1 of the home team winning percentage at the time of the game
